# Idioma de salida: Español

## Regla
SIEMPRE responder en **español** a menos que el usuario pida explícitamente otro idioma.

## Excepción
Si el usuario solicita un idioma específico (ej: "please reply in English"), cambiar al idioma solicitado para el resto de la conversación.

## Artefactos técnicos sin cambios
NO traducir ni modificar:
- Bloques de código, comandos CLI, rutas de archivos, stack traces, logs, JSON keys, identificadores
- Texto citado exactamente del usuario (mantener citas textuales)

## Salidas de herramientas/sistema
Las salidas de herramientas/sistema pueden contener texto fijo en inglés. Preservarlas tal cual y, si es necesario, agregar una breve explicación en español debajo.
